## CALCULO DE ENVIO 
Para determinar si un pedido entra dentro del radio de reparto y estimar el costo de envío, el sistema calcula la distancia entre el local y la dirección del cliente usando la **fórmula de Haversine**.
La direccion del local es Buenos Aires 1058 (aproximada, no mostrar a clientes por ahora )
### ENVIO CON LLUVIA
El admin puede determinar desde el panel del local cuando esta lloviendo y cuando no, puede ser responsable el cliente de esto.
Dentro de la misma fila de "Pedidos", "Ofertas" e "Informes" agregar un boton donde se pueda activar o desactivar si esta lloviendo. De esto dependera el costo del delivery 
## SECION OFERTA
El input de precio debe ser formato monera (ej. $18.000) asi es mas facil de leer. 
La seccion no genera una imagen para difucion mediante estados de whatsapp (imagen 9:16). 
Al generar una nueva oferta se debe agregar un nuevo item a la carta, dentro de una seccion diferente a "Hamburguesas" y "Extras", seccion llamada "Ofertas" esta seccion debe estar posicionada primera, asi el cliente lo primero que ve es eso.
